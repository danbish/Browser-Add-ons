The addon handler is in the lib/main.js file. Mostly used to communicate between panel and current tab.

The data folder contains all of the other files necessary to work.
accessible.js is the javascript that gets attached to the active tab to run test and modify DOM.
the panel folder contains the html and javascript for controlling whats displayed in the panel.
the report folder contains html and javascript for displaying and populating the report.

Ignore the test folder no tests have been set up.
